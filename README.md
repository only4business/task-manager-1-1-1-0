# TASK MANAGER

A project built with NovaKit Builder.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:5173 in your browser.

## Built With

- React + Vite
- Tailwind CSS
- TypeScript
